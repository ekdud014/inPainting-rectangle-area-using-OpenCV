InPainting rectangle area using OpenCV
============
Author: Matrivian

Usage:
----------------
Mouse:

    left button select rectangle area on the original image

Keyboard:

    press SPACE or "i" to inpaint image

    press "r" to restore inpainted image


COMPILE
----------------
./compile.sh

RUN
----------------
./inpaint imagename
